BBBS Information

Projects and Information related to the BBBS Application

Tools:
Navigation: Home | pkg-bbbs | BBBS-MRTG | bconfhtm

This project is for projects and other packaging related to BBBS, which is a complete BBS package with WWW/InterNet/modem/ISDN/LAN based access modes; see the BBBS home page for details regarding the BBBS application itself.

  • pkg-bbbs - The installer package bbbs-installer downloads the archive from the www.bbbs.net web site, or uses the one in the tmp directory if already present, then installs the application to the system. The package bbbs-equivs is for when you have manually installed BBBS and would like to make it like a standard Debian installed package.
  • BBBS::MRTG - A Perl Module which can be used to collect statistical information from an operational BBBS system, in order to be able to use such information with programs like MRTG. (The Perl Script mrtgbbbs.pl is an example of using the module.)
  • bconfhtm - The Perl script bconfhtm.pl can be used to create an html page from a file resulting from the BBBS command bbbs bconf and an HTML template file.
  • pkg-bterm - There is also the BTERM program from the same authors; packaging for that has been started but is not very far along yet.
