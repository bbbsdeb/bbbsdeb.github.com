This part of the site is for information related to Synchro BBS, which is a complete BBS package which allows you to "create a custom online service supporting multiple simultaneous users with hierarchical message and file areas, multi-user chat, and the ever-popular BBS door games". See its home page for details regarding the SBBS application itself.
